# GitHub Pages 配置步骤（图文版）

## 第1步：进入仓库设置
在GitHub仓库页面，点击顶部菜单中的 **Settings**

## 第2步：找到Pages选项
在左侧菜单中，找到并点击 **Pages**

## 第3步：配置Pages
在 **Build and deployment** 部分：

```
Source（来源）
❌ GitHub Actions
✅ Deploy from a branch  ← 选中这个

Branch（分支）
Branch: main  ← 确保是main
Folder: /(root)  ← 确保是/(root)

点击 [Save]
```

## 第4步：等待部署
- 页面会显示："Your site is live at ..."
- 部署通常需要1-3分钟
- 刷新页面直到链接出现

## 📱 手机安装步骤（重要！）

### iPhone安装：
1. 用Safari浏览器打开上面的链接
2. 点击底部的 **分享按钮** (方框向上箭头)
3. 滑动找到 **"添加到主屏幕"**
4. 点击 **添加**
5. 返回桌面，看到App图标

### Android安装：
1. 用Chrome浏览器打开上面的链接
2. 点击右上角的 **三个点** (菜单)
3. 找到 **"添加到主屏幕"**
4. 点击 **添加**
5. 返回桌面，看到App图标

---

## ⚠️ 重要提醒

1. **必须用手机浏览器打开**，不能用微信内置浏览器
2. iOS必须用Safari，Android必须用Chrome
3. 添加到主屏幕后，断网也能使用
4. App名称显示为"超声词汇"
